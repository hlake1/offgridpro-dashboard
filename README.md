# OffGrid Pro — Client Dashboard Mock

Prototype monthly performance dashboard built by Tweak Marketing.

**Live:** https://hlake1.github.io/offgridpro-dashboard/

## What it is

A visual mock of an automated Google Ads reporting dashboard for OffGrid Pro. Shows what a monthly client report could look like as a live web dashboard rather than a static slide deck.

## Features

- Executive summary KPIs (spend, leads, pipeline value, CPL)
- 6-month performance trend chart
- Campaign performance breakdown
- Top keywords + geographic lead origin
- Device split, search impression share, quality score
- **"What Tweak Did This Month"** section — makes the agency's value explicit
- Recommendations for the next reporting period
